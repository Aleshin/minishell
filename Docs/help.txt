Разрешенные функции:
readline: Считывает строку текста из стандартного ввода.
rl_clear_history: Очищает историю команд.
rl_on_new_line: Сообщает readline о начале новой строки.
rl_replace_line: Заменяет текущую строку в буфере readline на новую строку.
rl_redisplay: Перерисовывает строку в буфере readline.
add_history: Добавляет строку в историю команд readline.
printf: Выводит форматированную строку на стандартный вывод.
malloc: Выделяет блок памяти заданного размера.
free: Освобождает ранее выделенный блок памяти.
write: Записывает данные в файл или файловый дескриптор.
access: Проверяет доступность файла по указанному пути.
open: Открывает файл и возвращает файловый дескриптор.
read: Читает данные из файла или файлового дескриптора.
close: Закрывает файловый дескриптор.
fork: Создает новый процесс путем дублирования текущего процесса.
wait: Ожидает завершения дочернего процесса.
waitpid: Ожидает завершения конкретного дочернего процесса.
wait3: Ожидает завершения дочернего процесса и собирает статистику использования ресурсов.
wait4: Ожидает завершения конкретного дочернего процесса и собирает статистику использования ресурсов.
signal: Устанавливает обработчик для сигнала.
sigaction: Изменяет действие, которое происходит при получении определенного сигнала.
sigemptyset: Инициализирует набор сигналов как пустой.
sigaddset: Добавляет сигнал в набор сигналов.
kill: Отправляет сигнал процессу.
exit: Завершает выполнение программы.
getcwd: Получает текущий рабочий каталог.
chdir: Изменяет текущий рабочий каталог.
stat: Получает информацию о файле по его имени.
lstat: Получает информацию о символической ссылке по ее имени.
fstat: Получает информацию о файле по его файловому дескриптору.
unlink: Удаляет имя файла (ссылка на файл).
execve: Заменяет текущий процесс новым процессом.
dup: Дублирует файловый дескриптор.
dup2: Дублирует файловый дескриптор на заданный номер.
pipe: Создает канал для межпроцессного взаимодействия.
opendir: Открывает каталог для чтения.
readdir: Считывает следующую запись из открытого каталога.
closedir: Закрывает каталог.
strerror: Возвращает строку с сообщением об ошибке по коду ошибки.
perror: Выводит сообщение об ошибке на стандартный вывод.
isatty: Проверяет, является ли файловый дескриптор терминалом.
ttyname: Возвращает имя терминала, связанного с файловым дескриптором.
ttyslot: Возвращает номер терминала, связанного с текущим процессом.
ioctl: Выполняет управление устройством.
getenv: Получает значение переменной окружения.
tcsetattr: Устанавливает параметры терминала.
tcgetattr: Получает параметры терминала.
tgetent: Инициализирует терминал в библиотеке termcap.
tgetflag: Получает значение флага терминала.
tgetnum: Получает значение числового параметра терминала.
tgetstr: Получает строку, связанную с параметром терминала.
tgoto: Вычисляет управляющую строку для перемещения курсора.
tputs: Выводит строку управления терминалом.
